# SEG3525 Devoir 2 - Atelier Nomade

Prototype haute fidélité pour un site de service à rendez-vous.

## Service choisi

Atelier Nomade est un service mobile de réparation de vélos à Ottawa. Le site permet à un utilisateur de:

- comprendre le nom et le type d'entreprise;
- voir les coordonnées, les heures et les zones couvertes;
- consulter les services et les prix;
- réserver un rendez-vous interactif;
- voir le crédit du concepteur.

## Technologies

- React 18 via CDN
- Babel Standalone pour le prototype statique
- HTML/CSS/JavaScript
- Aucune dépendance locale requise

## Lancer localement

Ouvrir `index.html` directement dans un navigateur.

Option avec serveur local:

```bash
python3 -m http.server 5173
```

Puis ouvrir:

```text
http://localhost:5173
```
